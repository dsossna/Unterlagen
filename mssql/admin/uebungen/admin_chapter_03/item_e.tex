\item Erstellen Sie die in der folgenden Auflistung aufgeführten gMSAs in der
Organisationseinheit \oscommand{IT-SysAdmin\textbackslash
EXER\textbackslash Serverxx\textbackslash MSAs} für die Dienste Ihrer SQL Server-Instanz
ein. Ersetzen Sie dabei \enquote{xx} durch Ihre Platznummer!
\begin{itemize}
    \item \identifier{mssqlsrvrxx} für den SQL Server-Dienst
    \item \identifier{mssqlagentxx} für den SQL Server Agent
    \item \identifier{msdtssrvrxx} für den SQL Server Integration
    Services 12.0 Dienst
    \item \identifier{msrptsrvrxx} für den SQL Server Reporting Services
    Dienst
    \item \identifier{mssqlbrwsrxx} für den SQL Server-Browser
\end{itemize}
